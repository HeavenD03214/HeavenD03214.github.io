本地部署Hexo并进行配置

安装

详情参考：文档 | Hexo

  • Git

  • Node.js

  • Hexo

    • $ npm install -g hexo-cli //全局安装
$ npm install hexo //局部安装

  • 清除部署缓存并开启本地服务

    • hexo clean
hexo server
//访问localhost:4000可以看到hexo的helloWorld博客界面

  • 修改 hexo 配置文件_config.yml

    • theme:主题名称

$ npm install hexo-deployer-git –save

  • //博客根目录执行
hexo clean //清除旧的部署文件
hexo deploy //一键部署
